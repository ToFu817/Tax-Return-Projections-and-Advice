# Tax-Return-Projections-and-Advice

🇹🇼 台灣營所稅申報試算機 (AI 顧問版)

這是一個專為台灣企業主與會計從業人員設計的簡單試算工具。它能一次比較營所稅的四種申報方式，並透過 Google Gemini AI 評估**「查帳風險」與「成本費用是否充足」**，給予專業的申報策略建議。

✨ 特色功能

四種申報方式對比：核實申報(查帳)、擴大書審、所得額標準、同業利潤標準。

AI 風險評估：

當實際獲利過高，AI 會警示「書審風險」。

當成本費用不足，AI 會建議採用「所得額標準」申報以避免國稅局調帳。

隱私安全：所有數據皆在瀏覽器端運算，AI Key 儲存在您本地的 LocalStorage 中。

響應式設計：支援手機與桌面瀏覽器。

🚀 如何使用

取得 API Key：前往 Google AI Studio 免費申請 Gemini API Key。

設定 Key：開啟本工具，點擊右上角「設定 API Key」並填入。

查詢標準：至財政部網站查詢您行業的「擴大書審率」與「所得額標準」。

開始試算：輸入營業額與成本，AI 將自動為您分析最佳路徑。
